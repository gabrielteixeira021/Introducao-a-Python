Exercício 3 – Condicional Encurtada

Crie duas variáveis (a e b) com valores numéricos.
Use a condicional encurtada (if em uma linha) para mostrar qual número é maior.
Exemplo:

``print("A é maior") if a > b else print("B é maior")``
